# What Is Inside the Full Launch Kit

The full **OpenClaw Telegram Assistant Launch Kit** is a self-serve setup package for technical users who want a private AI assistant in Telegram without wasting days on setup mistakes.

It is designed to answer the practical questions that usually slow people down:

- What should I set up first?
- Should I start on my PC or VPS?
- Which model path is sane for day one?
- How do I avoid Telegram bot/config mistakes?
- How do I make the assistant feel useful instead of generic?
- What do I check when something breaks?

## Starter builds

The kit includes recommended starting architectures:

1. **Personal PC + Codex auth + Telegram + Personal Assistant**
   - best for a fast first useful assistant.

2. **Personal PC + OpenRouter + Telegram + Technical Copilot**
   - best for technical users who want flexible model access.

3. **VPS + direct API + Telegram + Founder / Operator**
   - best after you know you want uptime and remote reach.

These exist to reduce decision fatigue and prevent overbuilding on day one.
