## Starter builds

The kit includes recommended starting architectures:

1. Personal PC + Codex auth + Telegram + Personal Assistant
2. Personal PC + OpenRouter + Telegram + Technical Copilot
3. VPS + direct API + Telegram + Founder / Operator

These exist to reduce decision fatigue.

## Who should buy it

The kit is best for technical users who want:

- their own private AI assistant;
- Telegram as the interface;
- a structured OpenClaw setup path;
- model choice guidance;
- reusable persona templates;
- fewer setup mistakes.

It is not a done-for-you service and not a one-click SaaS.
